Tokenize Text

There are several tokenizer classes offered in the PHP Text Analysis package. Each tokenizer does slightly different than the others.

General Tokenizer

The GeneralTokenizer comes with a default set of charactorers to split on, they are: (white space) \n\t\r,—!?. You can customize the list by passing a string into the constructor with the characters you wish to split on 

            // Generic Tokenizer
            $tokenizer = new GeneralTokenizer();
            $tokens = $tokenizer->tokenize("This has some words");                        
            // Generic Tokenizer with custom split list
            $tokenizer = new GeneralTokenizer("|");
            $items = $tokenizer->tokenize("item1|item2|item3");
