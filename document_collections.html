Document Collections

Document Collections let you apply transformations and stemmers easily to all the documents that are going to be assessed. In this example, we will use the DocumentArrayCollection. The DocumentArrayCollection class lets you do zero based indexing or associative key based indexing

How to Create a Document Collection

The example below shows how to create a document collection and normalize it by setting all the tokens to lower case and removing the English stop words, and applying the Lancaster stemmer. 

            $books = [
                'poisonbelt' => 'https://ia801408.us.archive.org/10/items/poisonbelt00doylrich/poisonbelt00doylrich_djvu.txt',
                'kingsolomon'=> 'https://ia801406.us.archive.org/6/items/kingsolomonsmi00hagguoft/kingsolomonsmi00hagguoft_djvu.txt'
            ];
            // Generic Tokenizer
            $tokenizer = new GeneralTokenizer();
            $bookCollection = [];

                // create the raw token document

                // key based indexing
                $bookCollection[$title] = $document;
                // zero based indexing 
                //$bookCollection[] = $document;

            }
            $documentCollection = new DocumentArrayCollection($bookCollection);
            // apply transformers/filters
            $documentCollection->applyTransformations([new LowerCaseFilter(), new EnglishStopWordsFilter()]);
            // apply stemmer
            $documentCollection->applyStemmers([new SnowballStemmer()]);
