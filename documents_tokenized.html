Tokenized Documents

There are two types of documents available token documents and word documents. TokensDocument is a class that holds an array of tokens. The class allows you to apply transformations such as filters and stemmers to the tokens that make up the document. Upon creation of the document you can pass in a zones variable that may contain additional metadata about the document, such as creation data, isbn etc..

TokensDocument

The example below shows how to tokenize a document and normalize it by setting all the tokens to be lower case. 

            // Generic Tokenizer
            $tokenizer = new GeneralTokenizer();
            // create the raw token document
            $document = new TokensDocument($tokenizer->tokenize("This has some words"));  
            // apply the transformation to the document
            $document->applyTransformation(new LowerCaseFilter())
            // all the tokens should be lower cased
            var_dump($document->getTokens()[0]);
